

1) Multiprocessing Extenstions are an OPTIONAL set of extensions to the ARMv7-A
and ARMv7-R profiles, that provides a set of features that enhance
multiprocessing functionality.

2) The Large Physical Address Extension requires implementation of the
Multiprocessing Extensions.

3) The Multiprocessing Extensions obsolete the FCSE. This means that any
processor that includes the Multiprocessing Extensions cannot include the FCSE.
